🔄 Clear Y7 Foods Cache

If the website isn't loading properly, click the button below to clear all cached data and reload the site.

What this does:

  1. Unregisters all service workers
  2. Clears all cached files

Manual Hard Refresh:

  1. Windows: Press Ctrl + Shift + R or Ctrl + F5
